# Contributing

## Guidelines

1. Only submit tools that are **free to use** without mandatory signup
2. Tools should work **in the browser** — no installation required
3. Include a brief description of what the tool does
4. Maintain alphabetical order within sections

## Format

```
- [Tool Name](https://example.com) - Brief description of the tool.
```

## Pull Request Process

1. Fork the repository
2. Add your tool to the appropriate section
3. Submit a pull request with a clear description
